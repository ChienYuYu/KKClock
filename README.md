## 使用技術、套件

|  技術/套件   | 簡述  |
| :----- | :----- |
| Vue Cli 3  | 以 Vue.js 開發 SPA 網站 |
| VueRouter  | 設定路由 |
| Bootstrap 5  | 開發 UI 介面 |
| Vue-axios  | 串接 API |
| AOS  | 進場動畫 |
| Swiper  | 特價商品輪播 |
| vue-sweetalert2  | 彈出提示窗 |
